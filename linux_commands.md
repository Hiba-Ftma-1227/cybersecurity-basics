# Linux Commands Notes

## Introduction

Linux is an operating system widely used in cybersecurity, networking, and system administration. Kali Linux is a Linux distribution commonly used for penetration testing and security-related tasks.

---

# Basic Linux Commands

## pwd

Displays the current working directory.

Example:
pwd

---

## ls

Lists files and directories.

Example:
ls

---

## cd

Changes the current directory.

Example:
cd Documents

---

## mkdir

Creates a new directory.

Example:
mkdir cybernotes

---

## rm

Removes files or directories.

Example:
rm file.txt

---

## touch

Creates a new empty file.

Example:
touch notes.txt

---

## cat

Displays the contents of a file.

Example:
cat notes.txt

---

## clear

Clears the terminal screen.

Example:
clear

---

## cp

Copies files or directories.

Example:
cp file1.txt file2.txt

---

## mv

Moves or renames files.

Example:
mv old.txt new.txt

---

## chmod

Changes file permissions.

Example:
chmod 777 file.txt

---

## man

Displays the manual page of commands.

Example:
man ls

---

## ifconfig

Displays network interface information.

Example:
ifconfig

---

## ping

Checks network connectivity between systems.

Example:
ping google.com

---

## traceroute

Displays the route packets take to a network host.

Example:
traceroute google.com

---

## netstat

Displays network connections and statistics.

Example:
netstat

---

## whoami

Displays the current logged-in user.

Example:
whoami
